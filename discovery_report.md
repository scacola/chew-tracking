## Executive Summary

**한 줄 결론**: AirPods IMU 기반 저작·식사속도 트래킹의 *진짜 위험은 정확도가 아니라 "사용자가 자기 문제로 인식하지 않는다"는 점*이다 [사용자인사이트]. 따라서 진입한다면 "씹기 트래커" 도구로 팔지 말고, *결과(위 건강) 코치 + Apple이 흡수 못 하는 임상 콘텐츠·KOL·페르소나 자산*으로 진입해야 한다. **조건부 GO**, 단 *8주 검증 게이트(인터뷰 N=10 Q9~Q16 4개 이상 + 기술 프로토 + KOL 1명 영입 + 결제 30%)를 먼저 통과*해야 본격 빌드.

**핵심 권고**: **조건부 GO** — 신규 1순위는 **옵션 G "Chew & Calm Coach"** (Eat Right Now 모델의 한국 위염·소화불량 적용 — 임상 KOL + 28일 코스 + AirPods IMU 객관 데이터). 기존 옵션 B "Slow Bites"는 *G의 V1 측정 엔진 모듈*로 흡수, 옵션 F "ChewLens" (Discoveries 엔진)은 *G의 V1.5 모듈 + KOL 영입 실패 시 피벗 보험*. 옵션 D는 D2C 결제 실패 시 B2B 피벗. 처방형 PDT 직진(Pear $1.6B→$6M, 한국 솜즈 "현장 물음표")은 *명시적 함정*으로 거부. [제품-옵션G/F/H, 경쟁-보강B1/B3]

**의사결정의 3대 근거 (라운드 2 갱신)**:

1. **사용자 인사이트가 13개 직접 경쟁사 실패의 *근본 원인*을 짚었음**: HAPIfork·Vessyl·Bite Counter 등 직접 경쟁 13종이 모두 약했던 진짜 이유는 정확도/폼팩터가 아니라 *콘텐츠/임상/페르소나 자산이 없는 순수 도구*였기 때문이다 [사용자인사이트, 경쟁-보강B1]. Eat Right Now는 *측정 디바이스 없이도* 임상 RCT 1건 + Dr. Jud 개인 브랜드 + 28일 코스만으로 7년 살아남고 Sharecare에 인수됐다 [경쟁-보강B1]. 옵션 G는 거기에 *AirPods 객관 데이터 + 한국 KOL + 다노식 친근 페르소나 + 위 건강 결과 약속*을 결합한다.
2. **시장 + 기술 + 빈 공간이 *동시*에 정합**: 빠른 식사의 객관 페인 OR 1.5–2.15 + 한국 미란성 위염 +71% [시장-1.1·1.2], AirPods IMU 자유생활 F1 0.71–0.80 baseline [기술-2.1], 그리고 "임상 신경과학자 + 친근한 한국 페르소나 + 객관 데이터(IMU)" 트리오는 *완전 비어있는 White Space* [경쟁-보강B4]. 옵션 G가 1호 진입 가능.
3. **Apple 흡수 위험은 *임상 KOL/RCT/콘텐츠 IP*가 V1부터 자산일 때만 회피**: 2026.03 Apple jaw health 특허 [기술-1.2]는 *raw 데이터 OS 기능*만 흡수하고 *임상 KOL·한국어 위장 코스 IP·페르소나·RCT 누적·위 건강 의료 책임*은 흡수하지 않는다. 옵션 B는 자산이 *V1엔 없음(의도일 뿐)*, 옵션 G는 *V1부터 자산 빌드* — 사용자 인사이트 #2의 해법이 옵션 G에 명시 구현됐다 [제품-옵션G, 합성].

**즉시 다음 행동 1순위 (8주 검증 게이트로 갱신)**:
> **이번 주 안에 4트랙 병렬 시작**: ① 페르소나 1(한지원) + 페르소나 3(김상훈)형 *각 5명* 디스커버리 인터뷰 모집 — 신규 Q9~Q16 포함 16개 질문 풀 운영 [제품-옵션G 인터뷰], ② AirPods Pro 2세대 IMU 백그라운드 60분 + Now Playing API 부분 캡처 프로토타입 [기술-1.1, 5.6], ③ **한국 소화기내과·신경과학 KOL 5–8명 콜드 콘택트** (1명 자문 합의가 8주 게이트 필수 조건) [제품-옵션G 가설#4], ④ 한국 카피라이터·심리상담사 후보 1–2명 6개월 협업 의사 타진. 8주 후 *결제 30% + KOL 1명 + Q9~Q16 4개 이상 통과* 게이트로 본격 GO/NO-GO 재평가.

## 2. 기술 타당성 요약

**결론**: `CMHeadphoneMotionManager`로 AirPods Pro/3세대/Max IMU 합법 접근 가능. iOS 한정이며, *25Hz·단일채널·백그라운드 회색지대*라는 3대 API 제약이 있으나 본 시나리오에 우연히 맞물린다. [기술-1.1, 6.1]

**핵심 정량**:
- 학술 baseline: 실험실 F1 **0.86–0.91**(IMChew 2024, EarBit lab) / 자유생활 F1 **0.71–0.80**(EarBit wild). 짧은 간식만은 0.45–0.53로 급락. [기술-2.1]
- 음향+IMU 융합 시 0.97+이지만, AirPods raw 마이크는 일반 앱에 미노출 → **AirPods의 이점은 IMU 위주에 묶인다**. [기술-2.3]
- 25Hz Nyquist 12.5Hz는 일반 저작 빈도(0.94–2Hz)에 충분. *bruxism은 불가, eating chewing은 OK*. [기술-5.4]

**정확도 KPI 권고**: 임상이 아닌 *행동 변화 코칭용 F1 0.75–0.85*. ±10–15% 오차는 "오늘 평소보다 30% 빨리 먹었어요" 메시지에 영향 없음. [기술-6.1]

**4개 결정적 한계와 mitigation**:
1. 식사 외 활동(말하기·걷기) 분리 → *식사 시작 사용자 트리거* 또는 audio session/시간컨텍스트 활용. [기술-5.1]
2. 사용자별 보정(LOSO 5–25pp 차이) → *온보딩 1–2식사 캘리브레이션*. [기술-5.2]
3. 좌·우 동시 IMU 불가 → *chewing count 단일 채널 한계 안에서 가치 제안*. [기술-5.3]
4. 백그라운드 보장 미흡 → *영상 시청 audio session 활성으로 우회 — 본 시나리오와 우연히 잘 맞음*. [기술-5.6]

**Apple 흡수 신호 (위협)**: 2022 bruxism 특허 + 2026.03 jaw health metric 특허("좌·우 헤드폰 각각 bite/chew 카운트"). Apple 자체가 이 영역을 헬스 기능으로 보고 있음. [기술-1.2, 5.7]

| **Healbe GoBe** | 불가능한 약속 (손목 임피던스로 자동 칼로리 흡수) → 영구 신뢰 손상 | "자동으로 다 됨" 마케팅 위험. *측정만 약속, 추론은 보조*. |

### 4.3 현재 성공 모범 — Eat Right Now (Dr. Jud Brewer) [경쟁-3.4]

- 28일 마음챙김 식사 코스 + 임상 입증(craving-eating −40%, 부정 폭식 −36%) + 의료 라이선스 채널.
- *센서 없이 콘텐츠만으로* 성공.
- **시사점**: 행동변화 = 트래킹 ≠ 측정. *코칭 메시지·일일 미션*이 결합되어야 28일 코스. **Eat Right Now에 없는 = 실시간 객관 데이터** = 우리의 차별화.

### 4.4 5대 페인 패턴 (1-2점 리뷰 추출) [경쟁-4]

1. *"정확도 거짓말"* — 마케팅 약속 vs 실제 격차 = 신뢰 붕괴 근원
2. *"구독 사기성 + 코치는 챗봇"* — Noom BBB 1,200+ 환불 분쟁
3. *"백그라운드·동기 유지 실패"* — 14일 후 70% 이탈. **AirPods 자동 검출이 정확히 이 페인을 죽임**
4. *"내 식문화에 안 맞음"* — 한국 사용자 다수. 우리의 *씹기 자체*는 식문화 중립
5. *"감시받는 느낌, 죄책감 유발"* — 식이장애 위험. *친절한 톤 필수*

### 4.5 White Space — 가장 매력적인 빈 공간 3개 [경쟁-1차 5]

1. **백그라운드 자동 검출 + 혼밥 시청자 + 친절한 톤** (한국 1차 진입 정확)
2. **소화기/위식도역류/당뇨예방 의료 채널** (B2B 라이선스, 임상 근거 있음)
3. **Whoop/Oura의 회복 컨텍스트에 씹기 데이터 통합** (글로벌 헬스 옵티마이저)

### 4.6 보강 라운드 14건 핵심 학습 [경쟁-보강 B1·B2·B3·B4·B5]

라운드 2에서 임상 콘텐츠 코칭 8건 + 자기 인식 트리거 6건 분석 결과:

- **Eat Right Now (ERN)** — 측정 디바이스 *없이도* 임상 RCT 1건(craving-eating −40%) + Dr. Jud 개인 브랜드 + 28일 매일 강의 + Sharecare 인수로 *7년 생존*. 사용자 인사이트 #2(콘텐츠/페르소나 자산이 Apple 흡수 못 하는 해자)의 *가장 강한 증거*. [경쟁-보강B1]
- **Pear Therapeutics** — 처방형 PDT 1호 ($1.6B 정점) → 2023.4 파산 → $6M 자산 매각. 매출 $13.1M vs 비용 $136M (2022). *처방→약국→환자 워크플로우*가 보험사·의사·환자 누구도 적응 안 함 → **PDT 직진은 자금 소진 함정**. [경쟁-보강B1]
- **한국 솜즈/WELT-I** — 식약처 1·2호 인허가까지 갔으나 처방 후 실제 사용 미미 ("현장은 물음표"). **식약처 허가 ≠ 채택**. [경쟁-보강B3]
- **카카오헬스케어 파스타** — 한국 메이저가 *디바이스(CGM) + 콘텐츠 + 페르소나*로 진입한 *직접 모방 가능 모델*. 우리는 디바이스를 *사용자가 이미 가진 AirPods*로 대체. [경쟁-보강B1]
- **다노 (한국 모범)** — 임상 권위만으론 부족, *친근한 한국 인플루언서 페르소나*가 전환 깔때기. ERN형 임상 + 다노형 친근의 *결합*이 한국 최적. [경쟁-보강B1·B5#17]
- **Oura Discoveries** — 사용자가 의식 못한 행동 ↔ 결과의 *자동 상관 발견*. 우리는 *YouTube/Spotify 메타 ↔ 씹기 속도* 자동 상관으로 Oura보다 강한 Discoveries 가능. [경쟁-보강B2]
- **Lyra Health 2024–2025 흔들림** — 인간 코치 30 client/주 강제로 케어 품질 붕괴. 한국 인건비로 동일 함정 더 빠짐 → *AI 메인 + 한정된 인간 코치*. [경쟁-보강B5#12]

**권장 진입 경로 (라운드 2)**: D2C 코치 → 임상 RCT 누적 → B2B2C → 후행 처방. *처방 직진 금지*. [경쟁-보강B1·B3, 사용자인사이트#3]

### 4.7 보강 White Space — 콘텐츠+처방+자기인식 트리거 결합 [경쟁-보강B4]

기존 White Space에 *콘텐츠·임상·자기인식* 차원 교차 시 더 분명한 빈 공간:

1. **콘텐츠 결과 코치 (씹기 데이터 → 매일 카드)** — ERN(임상) + Noom(카드) + Oura(Discoveries) + 다노(친근) 결합. 씹기 데이터가 콘텐츠 *개인화 엔진*이 됨.
2. **위 건강·식후 컨디션 D2C 결과 코치** — 식이장애·다이어트 카테고리 회피 + 의료적 정당성 + 한국 위염 인구 막대.
3. **임상 신경과학자 + 친근한 한국 페르소나 + 객관 데이터(IMU) 트리오** — *글로벌 무비교*. 옵션 G 1호 진입 가능.

## 6. 핵심 모순·결정 포인트

리서치 4개 사이의 *충돌*을 노출한다. 각 모순은 곧 의사결정 포인트다.

### 모순 1: 기술은 가능하다 ↔ Apple이 OS에 흡수할 수도 있다

- **증거 A** [기술-1.1, 2.1, 6.1]: `CMHeadphoneMotionManager`는 합법 접근, F1 0.71–0.80 자유생활 baseline 존재. 본 시나리오 백그라운드는 영상 audio session으로 우회 가능.
- **증거 B** [기술-1.2, 5.7]: 2026.03 Apple "Jaw Health Metric Using Headphones" 특허 — 좌·우 헤드폰 각각이 bite/chew 카운트와 비대칭 측정. Apple이 OS Health 앱에 흡수할 가능성.
- **결정 필요**: Apple이 같은 기능을 *raw 메트릭*으로 OS에 내장하면 우리 차별화는 무엇인가?
- **결정 옵션**:
  - (a) **콘텐츠·페르소나 자산 우선 구축** (Eat Right Now 모델) — 측정은 Apple이 줘도 *코칭은 못 줌*. → 옵션 B의 길.
  - (b) **임상 인증·B2B 채널 우선** (Recovery Record 모델) — Apple은 의료기관 워크플로우를 만들지 않음. → 옵션 A·D 길.
  - (c) **로컬 특화** (한국 식문화·언어) — Apple은 글로벌 OS 회사. → 옵션 E 길.
- **권고**: (a) + (c)의 V2 흡수 = 옵션 B 메인 + 한식 모듈. (b)는 18–24개월 후. [합성]

### 모순 2: 객관 페인은 강하다 ↔ 주관 인식은 결과 매개일 뿐 *(라운드 2: 사용자 인사이트가 해법 제공)*

- **증거 A** [시장-1.1·1.2]: OR 1.5–2.15 메타분석 누적, 한국 미란성 위염 +71%. 의학적 "강"이 의심 여지 없음.
- **증거 B** [시장-2.1·2.3]: "씹기 횟수 트래킹" 자생 검색 거의 0. "알지만 안 함" 지배. 페인 인식이 *체중·소화기를 경유*해야만 들어옴.
- **증거 C (라운드 2 보강)** [사용자인사이트#1, 경쟁-보강B1]: **이게 13개 직접 경쟁사가 모두 실패한 *근본 원인*이다**. HAPIfork·Bite Counter·Vessyl이 측정만 하고 *깨달음 디자인 0*이었기 때문. ERN은 측정 디바이스 없이도 "Three Gears" 임상 프레임워크 + 28일 강의로 *내적 호기심 훈련*을 만들어 7년 살아남음.
- **결정 필요**: 사용자가 *스스로* 빠른 식사를 문제로 인식하지 않는데, 어떻게 매일 쓰게 만들 것인가?
- **결정 옵션**:
  - (a) **결과 KPI 직접 노출** ("위 건강 점수", "체중 정체 탈출 확률") — 옵션 B의 결과 프레이밍 대시보드.
  - (b) **의료 트리거** (위염 진단 직후 의사 처방) — 옵션 A. PDT 직진은 Pear/Somzz 함정 [경쟁-보강B1·B3].
  - (c) **마음챙김 라이프스타일 프레이밍** — 옵션 C. 페인 4/10로 retention 위험.
  - **(d) [라운드 2 신규] 깨달음 모먼트 + 임상 콘텐츠 결합** — *Discoveries 카드(옵션 F)가 자기 인식 트리거 → 28일 임상 코스(옵션 G)가 행동 처방 + 회복 서사*. 사용자 인사이트 #1·#2의 정합 해법. [경쟁-보강B2 Oura/Noom + B1 ERN]
- **권고 (갱신)**: **(d)**. 단독 (a)는 1차 보고서 옵션 B로서 *측정 도구* 함정에 가까움 [사용자인사이트#2]. 옵션 G가 (a)+(d)를 *결과 약속(위 건강) + Discoveries 자기 인식 트리거 + 28일 임상 코스* 한 제품에 통합. 13개 직접 경쟁사 실패 원인을 V1부터 해결한다. [합성, 사용자인사이트]

### 모순 3: 직접 경쟁사 13개 모두 약함/단종 ↔ 우리가 같은 함정에 빠질 위험

- **증거 A** [경쟁-1·2]: HAPIfork·Vessyl·Healbe·Bite Counter·20 Minute Eating·밀꼭 등 모두 좀비 또는 영세. *13종이 같은 페인을 풀려다 실패*했다는 사실은 시장 어려움 시그널.
- **증거 B** [경쟁-3·6]: 그러나 실패 원인이 *명확히 분류 가능*함 — 정확도 과장 / 폼팩터 마찰 / 콘텐츠 부재 / 카메라 의존 / 결과 부재. *각각 회피 가능한 함정*.
- **결정 필요**: 우리가 같은 함정을 피한다는 가설이 진짜인가, 자기 위안인가?
- **결정 옵션**:
  - (a) **MVP 출시 전 죽은 거인 패턴 체크리스트 통과 의무화** — 마케팅 약속 정직성 / 폼팩터 보유 디바이스 / 결과 KPI / 한식 호환 / 친절 톤. 5개 모두 충족 시 출시.
  - (b) **8주 베타 결제 게이트** — 4주차 결제 전환율 30%+ 미만이면 옵션 D 피벗 또는 가설 재구축. [제품-6.4]
  - (c) **포기** — 시장 진입 자체 안 함.
- **권고**: (a)+(b) 동시. 함정 회피 가설은 *행동 데이터*로만 검증되지 자체 분석으로 검증 안 됨. [합성]

### 모순 4: 백그라운드 자동 검출 = 차별화 ↔ 의식·동기부여 없으면 행동 변화 안 일어남

- **증거 A** [경쟁-WhiteSpace·페인#3][제품-2 옵션B 차별화]: AirPods 백그라운드 자동 검출은 "매 식사마다 앱 켜기" 14일 70% 이탈 페인을 죽인다. 제로 마찰 = 차별화 핵심.
- **증거 B** [경쟁-3.4 Eat Right Now][제품-2 옵션B MVP#3·#5]: 그러나 *측정만으로는 행동 안 바뀜*. Bite Counter가 무료인데도 안 뜨는 이유. Eat Right Now가 콘텐츠로 성공한 이유. 사용자가 *의식*해야 식사 속도가 변한다.
- **결정 필요**: "자동 검출(무의식)"과 "행동 변화(의식)"는 본질적으로 충돌한다. 어떻게 두 가지를 한 제품 안에 넣나?
- **결정 옵션**:
  - (a) **자동 측정 + *능동적 코칭 모먼트*** — 측정은 백그라운드, 그러나 *식사 중 햅틱*과 *주간 리포트 일요일*은 의식 트리거. 사용자가 *원할 때만* 의식. → 옵션 B의 MVP#3·#5 설계가 정확히 이걸 지향.
  - (b) **모드 전환** — 평소엔 자동, "오늘 의식 모드" 토글로 전환 시 능동 코치.
  - (c) **포기**: 자동 검출만 하고 행동 변화는 사용자에게 위임 — Bite Counter 함정 반복.
- **권고**: (a). 단, *햅틱 알림 수용성*이 페르소나 1·2에서 NO-GO 60% 이상이면 (b)로 fallback. 이는 인터뷰 가설 #1로 검증. [제품-6.1 질문#1·#8]

### 모순 5: 자기 투사 함정 ↔ 본 프로젝트 발의자가 핵심 페르소나

- **증거 A** [시장-거짓수요#4]: 발의자가 30대 IT 직장인 남성·혼밥+에어팟 시청 페인을 *직접* 경험 → 자기 시장을 보고 있을 가능성.
- **증거 B** [시장-페르소나#1·#2]: 그러나 1·2순위 페르소나(한지원 32세 IT 개발자, 박소연 29세 마케터)가 *발의자 인접 세그먼트와 정확히 겹침* — 즉 발의자의 페인이 *일반화 가능한 페인*일 가능성도 동시에 존재.
- **결정 필요**: 페르소나 1·2의 페인이 정말 발의자만의 페인이 아닌가?
- **결정 옵션**:
  - (a) **인터뷰 대상 다양화** — 한지원·박소연형 외 *반대 가설 그룹*(50대, 비IT, 비AirPods 사용자) 2–3명도 인터뷰에 포함.
  - (b) **결제 행동 게이트** — 설문 yes 응답 95%+이지만 결제 5%. 4주차 *실제 결제* 30%+ 게이트 [제품-옵션G]만 신뢰.
  - (c) **공동창업자/외부 자문** — 발의자가 아닌 사람의 시각 입력.
- **권고**: (a)+(b). 페르소나 검증은 *결제 행동*까지 가야 진짜. [시장-거짓수요#1·#4]

### 모순 6 (라운드 2 신규): 임상 콘텐츠 자산 빌드 비용 ↔ 8주 검증 속도

- **증거 A** [사용자인사이트#3]: 만들기 *전에* 8주 검증 (인터뷰 + 프로토 + 결제 30% 게이트). 빌드 우선·검증 후행은 Pear($1.6B→$6M, [경쟁-보강B1])·솜즈([경쟁-보강B3]) 함정.
- **증거 B** [제품-옵션G 리스크, 경쟁-보강B1 ERN]: 옵션 G의 *Apple 흡수 못 하는 자산* 핵심은 임상 KOL + 28일 한국어 코스 IP + 임상 RCT 누적 — 이게 V1부터 자산이어야 함. KOL 영입은 6개월, 콘텐츠 IP 제작은 6개월, RCT IRB 통과는 6–12개월 걸린다.
- **결정 필요**: 8주 게이트 통과 *전에* 어디까지 자산을 빌드할 것인가? 너무 안 만들면 게이트 통과해도 *V1 콘텐츠 비어있음*. 너무 만들면 게이트에서 NO-GO 시 매몰.
- **결정 옵션**:
  - (a) **8주는 *영입·자문 합의*까지만 (자산 *제작* 아님)** — KOL 5–8명 콜드 콘택트 → 1명 자문 합의 (서명 LOI). 카피라이터·심리상담사 *후보 1–2명 6개월 협업 의사 타진*. **실제 콘텐츠·RCT는 게이트 통과 후 시작**. 매몰 비용 최소.
  - (b) **8주에 *7일 미니 코스 1편*까지 빌드** — 옵션 H Phase 1 단순화 모드 [제품-옵션H]. KOL 1명 + 7일 미니 코스 + 측정 엔진 프로토 → 베타 N=20에 노출 → 28일 풀 코스 결제 의향 검증. 매몰 *중*.
  - (c) **8주에 *28일 풀 코스 + RCT 설계*까지 빌드** — Pear/Somzz 함정 직진. 매몰 *고*.
- **권고**: **(a) 우선, 게이트 통과 시 (b)로 즉시 전환**. 8주 게이트 = *결제 30% + KOL 1명 자문 합의 + Q9~Q16 4개+ + 기술 프로토 60분*. 4개 모두 통과 후에만 옵션 G V1 본격 빌드 시작 (28일 코스 + RCT 설계). [합성, 사용자인사이트#3]

## 8. 리스크 (라운드 2 갱신 — 우선순위 순)

리스크는 *지금 결정에 영향을 주는* 순서로 정렬. 결정 임팩트 ★★★ 순. **라운드 2 변경: Apple 흡수는 옵션 G의 임상 자산으로 *완화됨* / PDT 직진 함정이 새 ★★★ 리스크로 등장.**

### 리스크 1 (★★★ 신규): PDT 직진 자금 소진 함정 — *처방형 인증부터 받자는 결정이 사망 신호*

- **영향**: 식약처 디지털치료기기 인증을 V1 BM에 넣으면 1–2년 인증 기간 + 5억+ 비용 동안 매출 0. Pear Therapeutics는 매출 $13.1M vs 비용 $136M (2022)으로 2023.4 파산, $6M 자산 매각 [경쟁-보강B1]. 한국 솜즈/WELT-I는 식약처 1·2호 인허가까지 갔으나 처방 후 실제 사용 미미 ("현장은 물음표") [경쟁-보강B3] — *허가 ≠ 채택*.
- **발생 가능성**: 중. 본 프로젝트가 "의료적 정당성"에 끌려서 처방형 매력에 빠질 위험. 옵션 A GastroGuard가 직접 함정.
- **출처**: [경쟁-보강B1·B3, 사용자인사이트#3, 제품-옵션G 단계적 BM]
- **완화**:
  - **D2C 코치 → 임상 RCT 누적 → B2B2C → 후행 처방** 4단계 BM을 *문서화·공개 약속*. V1·V2 매출 검증 *전*에 인증 트랙 비활성.
  - 옵션 G의 Apple 흡수 방어 자산을 *콘텐츠/KOL/RCT*로 만들고, *식약처 인증을 자산으로 삼지 않는다*.
  - GO/NO-GO 프레임 6행("PDT 직진 함정 회피")을 *항상 GO 유지*해야 할 방어선으로 운영.

### 리스크 2 (★★ 라운드 2 완화): Apple jaw health OS 흡수 — *옵션 G의 임상 자산이 V1부터 빌드되면 완화*

- **영향**: raw 측정 차별화가 OS 기능에 흡수되면 *코칭 페르소나·콘텐츠*만 남는다. 그게 자산이 아니면 시장에서 추방.
- **발생 가능성**: 중-고. 2022년·2026년 특허 두 건 누적 → Apple은 "언제"의 문제. 12–24개월 가능. **단**, Apple은 *raw 데이터 OS 기능*까지만 흡수 — *임상 KOL·한국어 위장 코스 IP·임상 RCT·페르소나·위 건강 의료 책임*은 흡수 안 함 (Apple은 글로벌 OS, 한국 한정 위장 콘텐츠 IP 만들 인센티브 약함, 의료 책임 회피). [기술-1.2, 5.7, 제품-옵션G 차별화]
- **출처**: [기술-1.2, 5.7]
- **완화 (라운드 2 갱신)**:
  - **옵션 G의 V1부터 임상 자산 빌드** — KOL 1–2명 영입 + 28일 한국어 코스 IP + RCT 1편 (1년 내). 옵션 B(라운드 1)처럼 *V1엔 자산 없이 의도만* 두지 않는다.
  - Eat Right Now 모범([경쟁-보강B1]) — 측정 디바이스 *없이도* 7년 살아남고 Sharecare 인수. *콘텐츠·페르소나·임상 자산만으로 충분히 두꺼운 해자*임을 입증.

### 리스크 3 (★★★): 자유생활 정확도 부족 — *백그라운드 60분 미유지 시 핵심 가치 붕괴*

### 1주 내 — 4트랙 병렬 시작

1. **인터뷰 대상자 모집 N=10** — 페르소나 1형(한지원·위염) 5명 + 페르소나 3형(김상훈·대사증후군 경계) 5명. 지인망/링크드인/디시 슬로우이팅 후기 댓글/사람인 IT 직장인 커뮤니티 활용. entry filter = 헬스 앱 결제 경험 (Q16). *5일 내 일정 확정*. [제품-옵션G 인터뷰]

### 8주차 — 검증 게이트 의사결정

8. **8주 검증 게이트 평가 (Section 7 6행 통과 여부)** — 결제 30% + KOL 1명 + Q9~Q16 4개+ + 기술 프로토 60분 + 차별화 인식 + PDT 직진 회피.
   - **6행 모두 GO** → 옵션 G 본격 V1 빌드 시작 (6개월 빌드 — 28일 풀 코스 + 측정 엔진 + RCT 설계 + 페르소나 카드).
   - **5행 GO + KOL 행만 NO-GO** → 옵션 F (ChewLens) 단독 진입 피벗.
   - **결제 행 NO-GO 단독** → 옵션 D (ChewSync Pro) B2B SaaS 피벗 — 옵션 G 임상 RCT 자산을 영업 자산으로 전환.
   - **4행 이하 GO** → NO-GO 또는 가설 재구축.

### 게이트 통과 시 — 3개월 내 (옵션 G V1 빌드)

9. **카피라이터·심리상담사·KOL 6개월 협업 계약 시작** + **Closed Beta 빌드 N=20–30 모집** (옵션 G MVP 5개 기능 — 28일 코스 + 측정 엔진 + RCT 데이터 수집 + 페르소나 카드 + Discoveries 1개). [제품-옵션G MVP]
