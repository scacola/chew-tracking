## 마케팅 영향 분석

KOL 제거가 *옵션 G의 차별화 자산*에 미치는 영향:

| 자산 | 제거 전 | 제거 후 |
|------|---------|---------|
| (a) 한국 임상 KOL 자산 | ★★★ | — (제거) |
| (b) 임상 RCT 데이터 누적 | ★★ | ★★ |
| (c) 28일 한국어 코스 IP | ★★★ | ★★★ |
| (d) 친근한 한국 페르소나 코치 | ★★ | **★★★** (KOL 제거로 무게 이동) |
| (e) "위 건강 회복" 결과 라벨 | ★★ | ★★ |

**시사점:** 메인 차별화가 *KOL 권위*에서 *맥락 + 톤 + 임상 근거 투명성*으로 이동. 이는 디스커버리 보고서의 사용자 인사이트 #2(Apple이 흡수 못 하는 자산)와도 *일관*되며, 오히려 *현재 시점에서 더 정직한* 포지셔닝. 향후 실제 KOL 영입 합의 시 보강 라운드로 카드 추가 권장.
